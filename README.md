# kanji-checker-sheet-appscript

This is an app script code that automate Ctrl-Find a n3 kanji list to your existing review notes in your Google Spread Sheet.

# Inspiration

- Checking one by one using ctr-f to ensure that I have studied all n3 kanji list is tiring and prone to human error that's why I automate it.
- SRE eliminate toil by automating it so that's why I did this.
- When doing repetitive work, human error is prone so by doing this, I'm lessening human error.

- Go to your google spread sheet and click Extenstion > Apps Script
